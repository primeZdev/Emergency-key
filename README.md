This project creates an emergency subscription link for V2Ray configs.

## Installation

Run this command:

```bash
bash <(curl -s https://raw.githubusercontent.com/primeZdev/Emergency-key/main/install.sh)
```
After installation, paste your V2Ray configs with `em-keys edit-keys` (For add or change your v2ray configs).


## Subscription link
link: https://iran-domain/your_api_key

## Service Management

- Edit keys `edit-keys`
- Stop: `em-key stop`
- Restart: `em-key restart`
- Update: `em-key update`
- Uninstall: `em-key uninstall`

## Edit Configs

- Edit `.env` for PORT and API_KEY
- Add/edit configs in `keys.txt`
- Restart service after changes
